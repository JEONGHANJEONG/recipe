# 요리 레시피 관리 웹 서비스

## 1. 프로젝트 제목

**요리 레시피 관리 웹 서비스**

---

## 2. 주제 설명

이 프로젝트는 사용자가 자신만의 요리 레시피를 등록하고 관리할 수 있는 웹 서비스이다.
회원가입과 로그인을 통해 개인 레시피를 작성할 수 있으며, 다른 사용자의 레시피는 열람하고 댓글, 추천, 신고 기능을 사용할 수 있다.
관리자는 전체 레시피, 댓글, 신고 내역을 확인하고 관리할 수 있다.

---

## 3. 주요 기능

### 사용자 기능

* 회원가입 및 로그인 / 로그아웃
* 전체 레시피 목록 조회
* 레시피 상세 보기
* 레시피 검색 기능
* 카테고리, 난이도, 조리시간 필터 기능
* 보유 재료 기준 70% 이상 일치 레시피 필터 기능
* 레시피 등록, 수정, 삭제
* 본인이 작성한 레시피만 수정 및 삭제 가능
* 조리 방법에 사진 첨부 가능
* 댓글 작성, 수정, 삭제
* 추천하기 및 추천 취소
* 레시피 신고 기능

### 관리자 기능

* 전체 회원 목록 조회
* 전체 레시피 조회, 수정, 삭제
* 전체 댓글 조회 및 삭제
* 신고 목록 조회
* 신고 상태 변경

    * 접수
    * 처리중
    * 처리완료

## 5. 보안
SQL Injection 방어:
- 모든 SQL 값은 ? 파라미터 사용
- 검색어 LIKE도 ESCAPE 처리
  LIKE 검색에서 %, _ 문자가 와일드카드로 동작하지 않도록 처리.
  SQL Injection 방어는 ? 파라미터 바인딩으로 처리하고,
  이 함수는 LIKE 검색 조건 조작을 줄이기 위한 보조 방어.

XSS 방어:
- 템플릿에서 {{ }} 사용
- |safe 사용 금지

입력값 방어:  
사용자 입력값 기본 정리
- None 방지
- NULL 문자 제거
- 앞뒤 공백 제거
- 줄바꿈 제한 가능
- 최대 길이 제한
- 이메일 형식 검사
- 비밀번호 길이 검사
- 제목/댓글/재료/조리법 길이 제한
- 카테고리/난이도는 허용 목록만 사용
- 조리시간은 1~360 사이 숫자로 제한

## 6. 프로젝트 특징

이 웹 서비스는 기본적인 CRUD 기능뿐만 아니라 로그인 기반 권한 관리, 댓글, 추천, 신고, 관리자 페이지 기능을 포함한다.
또한 SQL Injection 방지를 위해 파라미터 바인딩을 사용하고, 입력값 검증을 통해 검색어, 회원가입 정보, 댓글, 신고 내용 등의 사용자 입력을 안전하게 처리한다.
